────────────────────────────────────────────
一、系統定位
────────────────────────────────────────────
跨模組「貨件流程監控 + AI 決策輔助」層，疊在鼎新 ERP iGP 之上。
- 鼎新 = 系統 of record（工單/庫存/BOM/採購數量以鼎新為準）
- 本系統 = 視覺化戰情 + 異常預警 + AI 解方
- 對鼎新唯讀不回寫，扣帳一律回鼎新操作

三、鼎新報表解析支援（17 種，自動辨識）

  INVR60  品號主檔（含條碼編號）

  INVR18  庫存數量表
  INVR19  進耗存統計表（期末庫存量）
BOM 類
  BOMR05  BOM 報表（多階）
製令類
  CSTR02  製令工時表（人時/機時）
  CSTR07  製令成本分析表
  CSTR08  製令成本明細表（含領料追溯）
  CSTR11  製令用料分析
  MOCR10  製令明細表（進度/狀態/急料）
  MOCR25  製令需求檢視表（缺料牆）
  MOCR43  製令未領料明細表（欠料牆）
領退料 / 委外
  MOCR11  領料明細表（批號/儲位）
  MOCR12  退料明細表
  MOCR14  託外進貨明細表（委外倉）
出貨 / 採購
  EPSR13  出貨通知明細
  IPSR02  採購進貨明細

七、部署 / 串接
────────────────────────────────────────────
- 技術：Next.js 16 + React 19 + Tailwind v4 + TypeScript
- 鼎新串接 3 種層級（詳見「鼎新直連配合清單」）
  · 方式 C 手動上傳報表（馬上可用，0 IT）
  · 方式 B 排程匯出資料夾（半自動）
  · 方式 A Linked Server 唯讀（即時，需 DBA）
- 全部唯讀不回寫，鼎新零風險
